# Use-Case: Electroplating

A simple but effective electroplating rig is shown in the example schematic below,
operated and monitored by a single Deimos DAQ.

![Electroplating use-case diagram](../assets/usecase_processing.svg){ .zoomable-image }

Etching and deposition baths can be organized similarly with chemistry and current control,
and plently of channels left over for additional features like temperature control,
colorimetry, and actuation of the workpiece or cathode.
